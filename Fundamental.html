FUNDAMENTAL JIU-JITSU PROGRAM
The Fundamentals Program is tailored for the beginning for practitioners. We will focus on the essential Jiu- Jitsu techniques and concepts that will build a solid self-defense and Jiu-Jitsu foundation.

The Fundamental Class is built specifically for new practitioners. The Fundamentals program will take you to the next level in a systematic and proven curriculum. We will outline exactly what is needed to progress from White to Blue Belt in our academy.
Fundamentals 1
(New students - No experience)

Fundamentals 1 is our beginner program for adults. In this program, you will learn the 36 core techniques of Gracie Jiu-Jitsu. Each one-hour lesson addresses one standing and one ground self-defense technique. You can start the program at any time and participate in any class without previous experience.

**Uniform (Gi) is required**
Fundamentals 2
(White Belt 3 stripes & up)

This class is focused on the Pedro Sauer Jiu-Jitsu White to Blue Belt Curriculum. We will expand upon the essential self-defense program to incorporate strategies and techniques to overcome more skilled adversaries. This class will require controlled positional sparring so completion of our Fundamentals 1 curriculum is advised.

**Uniform (Gi) is required**
